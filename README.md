--
*Dokumentasi dibuat untuk 4 sistem diantaranya :*
- Sistem Informasi Pengadaan Barang Milik Negara
- Sistem Pengarsipan Surat Menyurat
- Sistem Pengaduan Kerusakan Barang
- Sistem Informasi Kepegawaian
